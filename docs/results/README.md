# Public Results Snapshot

This folder contains a lightweight, publication-safe subset of the final dissertation results that can be published with the codebase.

Included here:
- single-split GAN comparison tables
- downstream full-pipeline comparison tables
- ablation tables
- cross-validation summary tables
- paired significance tests across the 5-fold runs
- selected training curves
- selected ablation figures

The original local dissertation workspace also contained raw per-case outputs, model checkpoints, generated STL bundles, and full evaluation directories. Those were excluded from the public repository because of size, licensing, and local absolute paths.
